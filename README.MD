# TempCtrl

A temperature controller based on Arduino with PlatformIO

## Hardware

- STM32F103C8T6
- SSD1306 128x32 OLED (IIC)
- MAX6675
- 5V relay module

## Software

- PlatformIO
- Arduino

## Pinout

PA0: MIN Button

PA1: ADD Button

PA2: MODE Buton

PA3: SCREEN Button

PA4: Relay EN

PA5: MAX6675 DO

PA6: MAX6675 CS

PA7: MAX6675 CLK

PB6: OLED SCL

PB7: OLED SDA
